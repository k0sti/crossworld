# cube

Core voxel octree data structure (Cube model).

Features:
- Octree-based voxel storage
- CSM (CubeScript Model) format parser
- Mesh generation with face culling
- Raycast operations for voxel picking
- MagicaVoxel (.vox) file loading
- Octree merge and optimization

The fundamental data structure for all voxel operations in Crossworld.
